Hint: You will probably see, very shortly that the name is added, the screen flickers and it's gone again. The problem is that it submits to the server, so the full page is reloaded. To prevent this you can call .preventDefault() on the event argument which will prevent the default behaviour (submit) and it should work. You can also use preventDefault() to prevent a link from actually forwarding to the link-address.
